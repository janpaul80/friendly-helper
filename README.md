 # HeftCoder AI Workspace
 
 A Vite + React + TypeScript application for AI-powered code generation.

## Getting Started

 Run the development server:

```bash
npm run dev
```

 Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

 ## Build

 ```bash
 npm run build
 ```
